# dev_Rust-Mac
Rust on Mac Notes

#### Install and check version
```
$brew install rust

## check via homebrew
$brew info rust

## check via rust
$rust --version

## Alternative install, which allows you to switch between rust versions without additional downloads
$brew install rustup
$rustup-init
$rustc --version

## Create project via cargo
$cargo new testproject

testproject
|-- Cargo.toml
|__ src
   |__ main.rs

```
